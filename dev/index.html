CleanHistograms.binweave ‚Äî Method
binweave(a)

Interleave the sequential values of a (<:AbstractVector) using the first and last values only once. To be used with interleave to make plotting-ready histograms.

Example

julia> binweave([1,2,3])
4-element Vector{Int64}:
1
2
2
3

CleanHistograms.quickhist ‚Äî Method
quickhist(x, binedges)

Quickly calculate histogram counts from collection x from an Abstract Range of binedges. Note that any values in x that fall outside of binedges will not be counted. Use at your own risk!
